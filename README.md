# Edwin Kihiuria — Portfolio

A clean, responsive single-page portfolio site showcasing projects, skills, and contact information.

## Features

- **Responsive Design** — Mobile-friendly layout with modern styling
- **Project Cards** — Showcase your work with descriptions and tech stack
- **Contact Form** — Built-in contact form with email support
- **Smooth Navigation** — Fixed navbar with scroll effects
- **Resume Link** — Direct download link to resume PDF

## Quick Start

1. Clone or download the repository
2. Open `Index.html` in your browser
3. Update the project entries and contact info as needed

## Files

- `Index.html` — Main site with all sections and inline scripts
- `style.css` — Responsive styles with dark theme
- `Avatar.PNG` — Profile image
- `Edwin_Kihiuria_Resume.pdf` — Resume file

## Customization

Edit project details in the Projects section, update contact email, and replace `Avatar.PNG` with your own image.

## Hosting

Deploy to GitHub Pages:
1. Push to a `gh-pages` branch or enable Pages in repo settings
2. Your site will be live at `https://yourusername.github.io/Portforlio`
